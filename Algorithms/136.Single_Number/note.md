# 2. 异或运算

- 异或运算的性质：
  - $a\oplus a=0$；
    - 相同值经异或运算得0；
    - 恰对应于每个相同的元素出现偶数次；
  - $0\oplus a=a$；
    - 任何值与0进行异或运算，均不改变该值；
  - 异或运算满足交换律与结合律：$a\oplus b\oplus a=(a\oplus a)\oplus b=0\oplus b =b$；
